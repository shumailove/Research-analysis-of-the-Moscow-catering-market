# Анализ рынка общественного питания Москвы
# Описание проекта
Исследование рынка общественного питания Москвы на основе открытых данных. Проект включает анализ локаций, типов заведений, ценовых категорий и популярности сетевых заведений.

# Цели исследования:
- Определить наиболее перспективные форматы заведений;
- Выявить популярные локации для открытия кафе/ресторанов;
- Проанализировать распределение сетевых и несетевых заведений;
- Оценить распределение цен по районам.

# Инструменты
- Python (анализ данных);
- Pandas (обработка и очистка данных);
- Matplotlib/Seaborn (визуализация);
- Jupyter Notebook (интерактивный анализ).

# Ключевые выводы
Топ-3 формата заведений:
- Кафе
- Рестораны
- Кофейни

Самые популярные районы:
- Центральный административный округ
- Юго-Западный административный округ

Средний чек:
- До 500 ₽ – 25% заведений
- 500–1000 ₽ – 45%
- Свыше 1000 ₽ – 30%

# Рекомендации:
- Открывать кофейни в ЦАО (высокая проходимость);
- Сетевые заведения эффективнее в спальных районах;
- Фуд-корты перспективны в торговых центрах.
